# Card Club - Online Card Games

Play Hearts & Spades and Hokm online with friends!

## How to Play Online

1. Clone this repository
2. Run `python server_final.py`
3. Access the game at `http://localhost:8766`
4. Share the link with friends on the same network

## Deploy to Railway

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template?template=https://github.com/yourusername/card-club)

## Features
- Play Hearts & Spades or Hokm
- Multiplayer support
- AI opponents
- Persistent game history
- Mobile responsive design
